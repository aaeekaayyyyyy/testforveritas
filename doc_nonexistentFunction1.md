# quickSort

## Orphan Documentation

This function implements the QuickSort algorithm to sort an array in-place.

### Parameters
- `arr` (number[]): The array to sort
- `low` (number): Starting index (default: 0)
- `high` (number): Ending index (default: arr.length - 1)

### Returns
- `void`: Sorts the array in-place, returns nothing

### Algorithm
Uses the divide and conquer approach with a pivot element.
